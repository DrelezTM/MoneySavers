## Installation 📑
* Clone Repository
  ```sh
  git clone https://github.com/DrelezTM/MoneySavers.git
  ```
* Open Directory MoneySavers
  ```sh
  cd MoneySavers
  ```
* Install Modules
  ```sh
  composer install
  ```
* Migration Database
  ```sh
  php artisan migrate
  ```
* Start Project
  ```sh
  php artisan serve
  ```
